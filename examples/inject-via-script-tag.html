render-telemetry demo

Open DevTools, run __renderTelemetry.start('demo'), interact with a React app served from this page (e.g. swap the script tag for your real bundle), then call __renderTelemetry.stop().

The floating toolbar at top-right gives the same actions without typing. Drag the ⋮⋮ handle to move it; ✕ hides it (Alt+Shift+R brings it back).
